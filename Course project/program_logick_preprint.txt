Функционал:
1) Библиотека должна содержать классы, которые реализуют упорядоченный список очередей:

2) как Список(LinkedList Связанный двунаправленный список) эта структура данных должна иметь следующий функционал:
 - иметь динамическую длинну
 - осуществлять автосортировку после добавления или изминения очередей( в качестве параметра сортировки будет выступать индекс) 
 - иметь индекс элемента(поле класса Node)
 - метод добавление(только в конец)+
 - метод удаления(с начала, с конца, по индексу)
 - метод выведения информации об элементе(по индексу)
 - метод выведения информации о всех элементах списка+
 - метод возвращения размера списка+
 - метод проверки на пустой список(isEmpty)+
 
 3) как Очередь (Queue)
 - метод добавления в очередь +
 - метод возвразения размера очереди + 
 - метод возврата элемента без удаления +
 - метод возврата элемента с удалением(из начала) +
 - метод удаления элемента (из начала)
